# 🏕️ WanderLust - Airbnb Clone (MERN Stack)

A full-stack web app for exploring and booking listings, inspired by Airbnb. Built with MongoDB, Express.js, React.js, Node.js, and EJS.

## 🔧 Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- EJS (templating)
- Cloudinary, Mapbox, Passport.js
